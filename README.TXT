======================== DESCRIPCION DE LA APLICACION =======================

La aplicación desarrollada para simular la calculadora de Karatsuba presenta al usuario una interfaz sencilla de utilizar y en un ambiente gráfico de ventana.
Para acceder a la aplicación, se ha creado un archivo ejecutable (Jar) el cual se encuentra en la carpeta Ejecutable y ha sido llamado "Karatsuba.jar", al ser ejecutable se inicia en un ambiente windows mediante el mouse (doble click) o por teclado (enter).
Si se desea ejecutar la aplicación mediante la consola de comandos, se debe ejecutar la siguiente instrucción: 
java -jar "\\Ruta_donde_se_almaceno\Ejecutable\Karatsuba.jar"

Ejemplo: java -jar "C:\Users\Documents\Ejecutable\Karatsuba.jar"
en el ejemplo anterior, la ruta donde se copió la carpeta descargada es: C:\Users\Documents\. 

Los requerimientos necesarios para ejecutar la aplicación son:
Sistema operativo Windows (XP, Vista, Seven)
Tener instalada la máquina virtual de Java (JVM)
Para obtener la aplicación, basta con descargar la carpeta Karatsuba y alojarla donde el usuario considere pertinente.
